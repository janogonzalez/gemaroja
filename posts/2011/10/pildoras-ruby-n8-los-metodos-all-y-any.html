Píldoras Ruby Nº8: Los métodos all? y any?

October 11 2011, 3:37 PM  by Jano González

Hoy revisaremos dos métodos presentes en las clases que incluyen el módulo Enumerable. El método all? nos indica si todos los elementos cumplen con una determinada condición, mientras que el método any? nos indica si al menos uno de los elementos cumple la condición.

Por ejemplo con un Array:

# Produce el resultado false
[1,2,3,4,5].all? { |n| n % 2 == 0 }
# Produce el resultado true
[1,2,3,4,5].any? { |n| n % 2 == 0 }

O con un Range:

# Produce el resultado false
(1..5).all? { |n| n % 2 == 0 }
# Produce el resultado true
(1..5).any? { |n| n % 2 == 0 }
