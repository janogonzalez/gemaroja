Disponible Ruby 1.9.2-p290

July 18 2011, 2:52 PM  by Jano González

Desde este fin de semana está disponible la nueva versión de Ruby 1.9.2. El release resuelve varios bugs, entre ellos algunos memory leaks, segfaults, problemas con psych, problemas con tk, etc.

Si usas RVM, actualizarse es tan simple como bajar la última versión de RVM y actualizar Ruby 1.9.2

rvm get head
rvm reload
rvm upgrade ruby-1.9.2-p180 ruby-1.9.2-p290

Si usas Windows, RubyInstaller ya fue actualizado con la nueva versión.

Revisa el listado completo de cambios.

(vía Ruby Inside)
