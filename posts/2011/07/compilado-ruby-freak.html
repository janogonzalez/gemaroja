ennnnnd

July 19 2011, 3:06 PM  by Jano González

El feature 5054 fue abierto para solicitar que se incorpore una “mejora” al lenguaje Ruby, que evita cerrar repetidamente bloques de código.

Por ejemplo, el siguiente código:

module MyModule
  class MyClass
    def my_method
      10.times do
        if rand < 0.5 
          p :small
        end 
      end 
    end 
  end 
end

Quedaría como:

module MyModule
  class MyClass
    def my_method
      10.times do
        if rand < 0.5 
          p :small
        ennnnnd

Al parecer el ambiente de la RubyKaigi estuvo muy cargado al alcohol este año :P.

Otra locura de la RubyKaigi: Aaron Tenderlove Patterson, core commiter de Ruby y de Rails, llevando el concepto de Code Kata a un nuevo nivel en el trailer de su película.
