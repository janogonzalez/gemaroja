Píldoras Ruby Nº2: El método map

May 2 2011, 5:58 PM  by Jano González

Al momento de trabajar con objetos como arreglos y rangos, es común tener que aplicar una transformación sobre cada uno de los elementos. En Ruby todas las clases que incluyen el módulo Enumerable nos permiten utilizar el método map, que al ser aplicado a un objeto enumerable produce un arreglo con los elementos transformados.

Por ejemplo con un Array:

# Produce el resultado [1,4,9,16,25]
[1,2,3,4,5].map { |n| n*n }

También podemos hacer lo mismo con un Range:

# Produce el resultado [1,4,9,16,25]
(1..5).map { |n| n*n }

O con un Hash:

# Produce el resultado [1,4,9]
{ :a => 1, :b => 2, :c => 3}.map { |k,v| v * v }
