Píldoras Ruby Nº3: El método select

May 3 2011, 4:45 PM  by Jano González

En la píldora de ayer vimos el método map, hoy revisaremos otro método muy útil de las clases que incluyen el módulo Enumerable. El método select entrega todos los elementos para los que el bloque de código entregado no se evalúa como false.

Por ejemplo con un Array:

# Produce el resultado [2,4]
[1,2,3,4,5].select { |n| n % 2 == 0 }

O con un Range:

# Produce el resultado [2,4]
(1..5).map [1,2,3,4,5].select { |n| n % 2 == 0 }

Si lo aplicamos sobre Hash, producirá otro:

# Produce el resultado { :b=>2 }
{ :a => 1, :b => 2, :c => 3 }.select { |k,v| v % 2 == 0 }
